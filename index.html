Initial News Release

Author: pAxton

DevilDust is a Mod for Arma 3. It currently has been in development since April of this year, with some time off in between. Many basics features of the mod are in semi-working condition. Several 3D models have been added to
the game as long with textures. Some characters also have been re-textured.

Concept of the Game mode

The mod, DevilDust, is centered around generating income so you can purchase weapons and other unique supplies to the mod. We will be able to loot materials to create drugs and sell them to the users in game. The goal of this mod is to generate intense gun fights and tactical play with incentives to staying alive. There is a thirst, hunger and addiction system in development. Since I do not want to spoil the surprises I will leave it at that.
